JSM 2026: Gaussian Processes with PyMC

Implementation plan · 2026-07-09 · Four-hour continuing-education course · Version-pinned local delivery

Executive Summary

The course ships four hour-aligned marimo notebooks plus a pre-course environment check. Every fitted model uses vendored observed data: salmon recruitment, coal-disaster counts, GB electricity demand, Walker Lake concentration, and grouped pitcher spin rates.

A single Pixi lock supports Windows, Linux, Intel macOS, and Apple Silicon macOS. Core PyMC work is CPython-only; browser-only WASM is deliberately excluded because PyMC/PyTensor fitting is not a supported Pyodide path.

Notebook flow is observe → predict → code exercise → inspect. Posterior sampling is explicit and click-gated; release requires visible PPCs, zero divergences, rank-R-hat below 1.01, and bulk/tail ESS above 400 for reported scalar parameters.

Environment

Python 3.13, PyMC 6.1.0, ArviZ 1.2.0, marimo 0.23.13. One lockfile solves all four supported platforms.

Data boundary

All data are local release artifacts. A provenance and redistribution record is required for every file; notebooks have no HTTP path.

Notebook boundary

Each numbered marimo file is independently teachable. Testable builders live in single-function @app.function cells using one setup cell.

Compute boundary

Widgets drive only low-cost prior/covariance displays. MCMC is gated by run_button; Hour 3 never runs posterior MCMC.

Build the salmon primer/Marginal GP and coal Poisson Latent GP with prior checks, gated posterior inference, predictions, diagnostics, and exercises.

Contrast exact GP with HSGP m=[50], c=1.5, then rehearse every gated fit against runtime, diagnostics, and cross-platform acceptance gates.

Dependency: all prior phases and the final lockfile.

Canonical Model Contracts

PythonExact salmon Marginal GP
cov = eta**2 * pm.gp.cov.Matern52(1, ls=ell)
gp = pm.gp.Marginal(mean_func=pm.gp.mean.Constant(c=alpha), cov_func=cov)
gp.marginal_likelihood("recruits", X=x_data, y=y, sigma=sigma)
PythonElectricity kernel composition
cov_long = eta_long**2 * pm.gp.cov.Matern52(
    input_dim=2, ls=ell_long, active_dims=[0]
)
cov_year = eta_year**2 * pm.gp.cov.Periodic(
    input_dim=2, ls=ell_year, period=1.0, active_dims=[1]
)
gp = pm.gp.Marginal(cov_func=cov_long + cov_year)
PythonFull-series HSGP baseline
cov = eta**2 * pm.gp.cov.Matern52(1, ls=ell)
gp = pm.gp.HSGP(m=[50], c=1.5, cov_func=cov)
f = gp.prior("f", X=x_data, dims="obs", shape=len(y_full))
pm.Normal("demand", mu=f, sigma=sigma, observed=y_full, dims="obs")

Definition of Done

  • One Pixi lock installs and passes the environment check on Windows, Linux, Intel macOS, and Apple Silicon macOS.
  • Every distributed dataset has verified source, redistribution basis, schema, transformation, and deterministic subset tests.
  • All four marimo notebooks pass strict validation, expose their exercises, and execute non-sampling paths offline after bootstrap.
  • Each canonical posterior model exposes prior predictive, posterior predictive, divergence, rank-R-hat, and ESS outputs through DataTree access.
  • Release rehearsals meet every runtime, convergence, PPC, and HSGP-boundary gate on the stated laptop baseline.
